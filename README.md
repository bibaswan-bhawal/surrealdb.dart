## Quickstart-Guide

### Install

Import the library into your dart project with

```bash
dart pub add surrealdb.dart
```

or for a flutter project

```bash
flutter pub add surrealdb.dart
```

```dart
import 'package:surrealdb.dart/surrealdb.dart'
```
